PCML converter is an open source tool provided by infoview systems to convert IBM i (AS400) PCML formats to IBM i(AS400) program definition as well as RAML.


Once the PCML format converts into program definition and RAML, use copy to clipboard feature to copy the generated contents and These program definitions can be use to configure "AS400 program call connector" and RAML can be used to generate flows with respect to "AS400 program call connector" in MuleSoft. For RAML Examples can be generated automatically based on parameters type and can be separated as request and response examples with this feature it will be easy to understand the developers.
